# UC-StaffDuty

A simple staff duty system designed to improve roleplay, reduce staff interference, and help prevent staff misuse while on duty.

- 🔈 Staff Tags
- 👻  Ghost Mode (Passive Mode)
- 🛍️ Uses player Ace's
